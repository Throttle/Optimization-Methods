1) Как ведет себя простой симплекс при приближении к точке локального минимума?
	Простой симплекс должен закручиваться улиткой при подходе к точке локального минимума
2) Вы реализовали Метод случайного поиска с возвратом? Что это?
	Метод случайного поиска с возвратом: 
	Входные данные:
    		а) координаты базовой точки
		б) кол-во точек которые будут сгенерированы для проверки
		в) требуемая точность
		г) длина отрезка в пределах которого будут генерироваться точки
	Суть метода:
		Генерируем множество точек в окрестности базовой точки,
		вычисляем значение функции для каждо точки, как только удалость понизить
		значение функции - переходим в эту точку. Она становится базовой. И тд.
